# 🚀 [WhereAmI]

## 📖 Sobre
Projeto de evolução de uma ferramenta recriada por engenharia reversa, transformada em um produto autoral com foco em inovação e experiência do usuário.

---

## 🎯 Objetivo
- Criar uma solução original
- Melhorar UX/UI
- Adicionar funcionalidades relevantes
- Utilizar Firebase como infraestrutura

---

## 🎨 Design
- Estilo: Minimalista, cores clara, interface intuitiva.

---

## 🔥 Funcionalidades
- Historico utilizando o Cloud Firestore
- Exportação de relatorio em documento pdf
- Tela de login com integrando a conta do google
- Utilizando o Analytics, monitore onde os usuarios mais clicam

---

## ⚙️ Tecnologias
- Engenharia de Prompt
- IAStudio
- Firebase (Auth, Firestore, Hosting)
